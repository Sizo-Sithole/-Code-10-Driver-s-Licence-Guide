# How to Get a Code 10 Driver’s Licence (Heavy Motor Vehicle): A Step-by-Step Guide

A collaborative Markdown knowledge base that explains the process of getting a **Code 10 driver’s licence** for heavy motor vehicles.  
This repository is also a practical demonstration of clean **Git and GitHub collaboration**, including branching, pull requests, reviews, and tagging releases.

## What this project covers
This guide aims to document, in a clear beginner-friendly way:
- The typical journey from learner’s licence to a Code 10 driving licence
- Key requirements and documents
- Testing stages and what to expect
- Practical preparation tips
- Common mistakes and how to avoid them

## Disclaimer
This guide is for educational purposes. Licensing rules and procedures can differ by country, province, and may change over time.  
Always confirm details with your local traffic authority before applying.

## Target audience
- First-time applicants for Code 10
- Anyone upgrading from a smaller licence code
- Students building a knowledge base with Git/GitHub
