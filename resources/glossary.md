# Glossary

**Code 10**  
Commonly used term in South Africa for the medium heavy vehicle category.

**C1**  
Formal licence code category associated with Code 10.

**DLTC**  
Driving Licence Testing Centre.

**GVM**  
Gross Vehicle Mass.

**PrDP**  
Professional Driving Permit, required for certain types of professional driving.

**Yard Test**  
The controlled driving assessment done inside the testing yard before the road test.

**Roadworthy**  
A standard confirming that a vehicle is safe and legally fit to be on the road.
