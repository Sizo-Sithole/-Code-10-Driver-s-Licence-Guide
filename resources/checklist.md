# Code 10 Licence Checklist (South Africa)

Use this as a quick planning tool.

## Before your learner’s test
- [ ] Valid ID
- [ ] Required photos
- [ ] Proof of address (if required by your DLTC)
- [ ] Learner’s test booking confirmed
- [ ] Study plan completed
- [ ] Practice tests done

## Before your driving test
- [ ] Valid learner’s licence
- [ ] Driving test booking confirmed
- [ ] Correct Code 10 test vehicle arranged
- [ ] Vehicle roadworthy
- [ ] All documents copied and ready
- [ ] Pre-trip inspection practice completed
- [ ] Yard test practice completed
- [ ] Road test routes and rules reviewed

## After passing
- [ ] Licence issuing process started
- [ ] Temporary licence collected (if issued)
- [ ] PrDP application started (if needed for your work)
